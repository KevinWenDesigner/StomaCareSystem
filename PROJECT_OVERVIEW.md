# 造口护理系统 - 项目概览

## 📋 项目简介

造口护理系统是一个基于微信小程序的智能医疗护理平台，旨在帮助造口患者进行日常护理管理、健康监测和护理教育。系统集成AI图像分析技术，为患者提供专业的护理指导。

## 🏗️ 系统架构

```
造口护理系统
├── 患者端小程序 (patientApp/)
│   ├── 微信小程序框架
│   └── 前端界面和交互
│
├── 护士端小程序 (nurseApp/) - 待开发
│   ├── 微信小程序框架
│   └── 护理管理界面
│
└── 后端服务 (backend/)
    ├── Node.js + Express
    ├── MySQL 数据库
    └── RESTful API
```

## 🎯 核心功能

### 患者端功能

1. **AI智能评估** 📸
   - 拍照上传造口图片
   - AI自动分析造口状况
   - 风险等级评估
   - 个性化护理建议

2. **症状日记** 📝
   - 每日症状记录
   - 排泄物情况跟踪
   - 疼痛和不适记录
   - 数据统计分析

3. **护理教育** 📚
   - 分类课程学习
   - 视频教学内容
   - 学习进度跟踪
   - 操作指导

4. **健康报告** 📊
   - 数据可视化分析
   - 趋势预测
   - 健康评分
   - 改善建议

5. **护理计划** 📋
   - 个性化护理方案
   - 任务清单管理
   - 进度跟踪

6. **智能提醒** ⏰
   - 用药提醒
   - 更换造口袋提醒
   - 复诊提醒
   - 自定义提醒

7. **家属管理** 👨‍👩‍👧
   - 家属信息管理
   - 紧急联系人设置

### 护士端功能（规划中）

1. **患者管理**
   - 患者档案管理
   - 护理记录查看
   - 评估结果审核

2. **在线指导**
   - 远程咨询
   - 护理方案制定
   - 问题解答

3. **数据分析**
   - 患者数据统计
   - 护理效果评估

## 🗄️ 数据库设计

### 核心数据表（15张）

1. **用户相关**
   - `users` - 用户基础信息
   - `patients` - 患者档案
   - `nurses` - 护士信息

2. **健康数据**
   - `assessments` - AI评估记录
   - `symptom_diaries` - 症状日记

3. **教育内容**
   - `course_categories` - 课程分类
   - `courses` - 课程内容
   - `learning_records` - 学习记录

4. **护理管理**
   - `care_plans` - 护理计划
   - `care_plan_items` - 计划项目
   - `reminders` - 提醒设置
   - `reminder_logs` - 提醒记录

5. **其他**
   - `family_members` - 家属信息

## 🚀 快速开始

### 后端服务启动

```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
npm install

# 3. 配置环境变量
copy .env.example .env
# 编辑 .env 文件配置数据库

# 4. 初始化数据库
npm run init-db

# 5. 启动服务
npm run dev
```

### 小程序启动

```bash
# 1. 使用微信开发者工具打开 patientApp 目录
# 2. 配置小程序AppID
# 3. 编译运行
```
