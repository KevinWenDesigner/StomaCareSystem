## ✅ 快速检查清单

### 开发环境

- [ ] Node.js >= 14.0.0
- [ ] MySQL >= 5.7
- [ ] 已创建 `backend/.env` 文件
- [ ] 已初始化数据库 (`npm run init-db`)
- [ ] 后端服务运行中 (`npm run dev`)
- [ ] 小程序 `config.js` 配置为 `http://localhost:3000/api`
- [ ] 微信开发者工具勾选"不校验合法域名"

### 生产环境

- [ ] 服务器已购买
- [ ] 域名已注册并解析
- [ ] SSL证书已配置
- [ ] 后端代码已上传
- [ ] 数据库已创建
- [ ] PM2/IIS已配置
- [ ] 防火墙已开放端口
- [ ] 微信公众平台域名已配置
- [ ] 小程序 `config.js` 已更新为生产地址
