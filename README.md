# 造口护理智能管理系统

## 项目结构

本项目已重新构建为两个独立的小程序，分别针对患者端和护士端：

```
stoma-care-system/
├── patient-app/          # 患者端小程序
│   ├── app.json         # 患者端配置
│   ├── app.js           # 患者端逻辑
│   ├── app.wxss         # 患者端样式
│   └── pages/           # 患者端页面
│       ├── index/       # 欢迎页面
│       ├── home/        # 患者主页
│       ├── camera/      # AI评估
│       ├── diary/       # 症状日记
│       ├── education/   # 护理教育
│       └── profile/     # 个人中心
│
├── nurse-app/           # 护士端小程序
│   ├── app.json         # 护士端配置
│   ├── app.js           # 护士端逻辑
│   ├── app.wxss         # 护士端样式
│   └── pages/           # 护士端页面
│       ├── index/       # 欢迎页面
│       ├── dashboard/   # 工作台
│       ├── patients/    # 患者管理
│       ├── tasks/       # 任务管理
│       └── analytics/   # 数据分析
│
└── miniprogram/         # 原始混合版本（保留）
```

## 功能特点

### 患者端功能
- **AI智能评估**: 拍照获得专业评估建议
- **症状日记**: 记录每日症状变化
- **护理教育**: 学习专业护理知识
- **个人中心**: 管理个人信息和设置

### 护士端功能
- **工作台**: 查看患者状态和任务概览
- **患者管理**: 管理患者信息和状态
- **任务管理**: 分配和跟踪护理任务
- **数据分析**: 查看护理效果分析

## 技术特点

### 设计风格
- **现代化UI**: 采用Figma设计风格
- **渐变背景**: 使用蓝色渐变主题
- **圆角卡片**: 统一的卡片设计语言
- **动画效果**: 丰富的交互动画
- **响应式**: 适配不同屏幕尺寸

### 技术栈
- **微信小程序**: 原生开发
- **WXML/WXSS**: 页面结构和样式
- **JavaScript**: 业务逻辑
- **本地存储**: 数据持久化

## 页面结构

   - 今日状态统计
   - 最近记录
   - 护理提醒

3. **AI评估** (`pages/camera/camera`)
   - 拍照功能
   - AI分析结果
   - 专业建议

4. **症状日记** (`pages/diary/diary`)
   - 症状记录
   - 统计分析
   - 历史查看

5. **护理教育** (`pages/education/education`)
   - 课程学习
   - 知识库
   - 学习进度

6. **个人中心** (`pages/profile/profile`)
   - 个人信息
   - 设置选项
   - 版本信息

### 护士端页面
1. **欢迎页面** (`pages/index/index`)
   - 功能介绍
   - 护士登录
   - 品牌展示

2. **工作台** (`pages/dashboard/dashboard`)
   - 统计概览
   - 最近患者
   - 待办任务

3. **患者管理** (`pages/patients/patients`)
   - 患者列表
   - 搜索筛选
   - 患者详情

4. **任务管理** (`pages/tasks/tasks`)
   - 任务列表
   - 任务分配
   - 进度跟踪

5. **数据分析** (`pages/analytics/analytics`)
   - 护理效果
   - 统计图表
   - 趋势分析

## 部署说明

### 患者端部署
1. 在微信公众平台创建患者端小程序
2. 配置AppID和域名
3. 上传 `patient-app` 目录代码
4. 提交审核发布

### 护士端部署
1. 在微信公众平台创建护士端小程序
2. 配置AppID和域名
3. 上传 `nurse-app` 目录代码
4. 提交审核发布
